Ето кратък списък с най-важните Git команди, разделени по категории за бърза справка:
1. Първоначална настройка
git config --global user.name "[име]" – задава име на потребителя.
git config --global user.email "[email]" – задава имейл за комитите.
git config --list – показва всички текущи настройки. 
2. Създаване на хранилище (Repository) 
git init – създава ново локално хранилище в текущата папка.
git clone [url] – копира съществуващо отдалечено хранилище на вашия компютър. 
3. Работа с промени
git status – показва състоянието на файловете (променени, подготвени за запис или непроследявани).
git add [файл] – добавя конкретен файл в „staging“ зоната (подготовка за запис).
git add . – добавя всички променени файлове.
git commit -m "[съобщение]" – записва подготвените промени с описание.
git commit --amend – позволява да промените последния комит (напр. да добавите файлове или да коригирате съобщението). 
4. Разклонения (Branches)
git branch – извежда списък с всички локални клонове.
git branch [име] – създава нов клон.
git checkout [име] или git switch [име] – превключва към посочения клон.
git checkout -b [име] – създава нов клон и веднага превключва към него.
git merge [име] – слива посочения клон в текущия.
git branch -d [име] – изтрива локален клон. 
5. Синхронизация с отдалечено хранилище (Remote) 
git remote add origin [url] – свързва локалното хранилище с отдалечен сървър.
git push origin [клон] – изпраща локалните промени към сървъра.
git pull – изтегля и автоматично слива промените от сървъра в текущия клон.
git fetch – изтегля промените от сървъра, но без да ги слива веднага. 
6. Преглед на историята и отмяна
git log – показва историята на всички комити.
git log --oneline – показва историята в съкратен вид (по един ред на комит).
git diff – показва разликите между работната директория и последно записаните промени.
git checkout -- [файл] – отменя промените в конкретен файл и го връща към последното състояние от хранилището.
git reset --hard HEAD – изтрива всички локални промени и връща проекта към състоянието на последния комит.
